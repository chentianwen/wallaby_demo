# [WallabyDemo](https://github.com/wallaby-rails/wallaby-demo)

This is the demo repo for [Wallaby](https://github.com/wallaby-rails/wallaby) gem to showcase what Wallaby is capable of. Please visit [https://wallaby-demo.herokuapp.com/](https://wallaby-demo.herokuapp.com/) to give it a try.

The customization is done in the following folders:

- app/controllers/admin/
- app/decorators/admin/
- app/views/admin/application/home.html.erb

Feedback is welcome via [https://github.com/wallaby-rails/wallaby/issues/new](https://github.com/wallaby-rails/wallaby/issues/new)
